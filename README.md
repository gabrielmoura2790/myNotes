# 💻Projeto: myNotes

Aplicativo para você anotar suas atividades e marcar como concluídas.

# 🌟Tecnologias

## Front-End

- React Native Expo
- TypeScript
- AsyncStorage
- React Native Gesture Handler
- React Native Reanimated
- Vector Icons
- Styled Components

## Back-End

- NodeJs
- TypeScript
- Express
- Prisma
- PostgresSQL
- bcryptjs
- jsonwebtoken
- cors

# 🛠️Funcionalidades

- [x] Criar atividades
- [x] Deletar atividades
- [x] Marcar atividades como concluídas
- [x] Editar atividade
- [x] Login e cadastro de usuário
